# KNN-Model
Iris Flower Classification using K-Nearest Neighbors (KNN)

This project is a beginner-friendly data science exploration where I use the K-Nearest Neighbors (KNN) algorithm to classify iris flower species from the classic Iris dataset. The dataset is built into many Python libraries and is a common starting point for learning machine learning fundamentals.
